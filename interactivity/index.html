Arte Interattiva

Opere interattive con realtà aumentata

{% if page.carousel %} {% include carousel.html %} {% endif %}

Le mie opere di arte interattiva sono elencate in questa pagina. La maggior parte delle opere sono costituite da dipinti e disegni con realtà aumentata, vai alla mia pagina delle opere per scoprire di più.

Lavoro con progetti Open Source come AR.js, Jsartoolkit5, Three.js e altri. Tutti le app sono basate su linguaggi di programmazione HTML e Javascript. Non è richiesta l'installazione di nessun software aggiuntivo, è necessario solamente collegarsi alla pagina, cliccando sul link. Tutto ciò di cui avete bisogno è uno smartphone o un tablet e ... l'opera d'arte! Sì, non puoi fare tanto senza di loro. Potete comunque provare a scaricare le immagini, stamparle e collegarvi alla pagina corrispondente con il vostro smartphone o tablet per testare l'opera interattiva. Ma ho intenzione di realizzarne alcuni applicazioni che puoi scaricare, stampare un'opera d'arte e interagire con essa! Rimanete sintonizzati!
