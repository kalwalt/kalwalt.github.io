Arte Interattiva

Opere interattive con realtà aumentata

{% if page.carousel %} {% include carousel.html %} {% endif %}

Le mie opere di arte interattiva sono elencate qui. La maggior parte delle opere sono costituite da dipinti con realtà aumentata, vai alla mia pagina delle opere per scoprire di più.

Lavoro con progetti Open Source come AR.js, Three.js e altri. Tutti le app sono basate su linguaggi di programmazione HTML e Javascript. Tutto ciò di cui avete bisogno è uno smartphone o un tablet e ... l'opera d'arte! Sì, non puoi fare tanto senza di loro. Ma ho intenzione di realizzarne alcuni applicazioni che puoi scaricare, stampare un'opera d'arte e interagire con essa! Rimanete sintonizzati!
